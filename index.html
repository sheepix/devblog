安装

  • 安装Git
  • 安装Node.js

    安装Hexo

部署到远程站点

Github

  1. 创建github repo
    https://github.com/sheepix/devblog.git
  2. 修改hexo项目下的/_config.yml文件
  3. 在项目目录内安装git部署插件
  4. 执行部署

配置GitHub Pages

  • Source选项选择master分支
  • Custom domain设置成想要的名字如devblog.sheepx.tech
  • 在sheepx.tech域名解析内添加CNAME记录：
    将 devblog.sheepx.tech
    指向到 [github用户名].github.io
  • 在项目/source/文件夹下创建名为CNAME的文件并填入域名，如 devblog.sheepx.tech 这样才不会每次deploy都覆盖掉github上的设置
    GitHub Docs

后续发布文章

  1. hexo new [POST_TITLE] or hexo new post [POST_TITLE]
  2. 编辑/source/_posts/[POST_TITLE].md
  3. 执行

更换主题

  1. 在项目目录中
  2. 修改/_config.yml
  3. 重建
